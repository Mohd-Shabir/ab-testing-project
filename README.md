# A/B Testing Analysis: Snowflake · SQL · Python · Tableau

An end-to-end A/B testing project analysing the impact of a new webpage design on user conversion rates. The analysis covers data storage and querying in Snowflake, statistical significance testing in Python, and interactive dashboard visualization in Tableau.

**Key Finding:** Group B (treatment) achieved a **14.07% conversion rate** vs **5.40% for Group A** (control) — a **160% relative improvement** with extremely high statistical significance (p < 0.001).
